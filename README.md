# ARiSA - GPT가 탑재된 강력한 AI 비서

 다양한 플랫폼에서 여러분의 업무를 도와드릴 귀여운 AI 메이드 비서입니다.
                                                                                                                                                                                
## 디스코드 봇 실행

 `python run_discord.py`

### 헤로쿠 호스팅

 Procfile이 이미 들어 있습니다. 그대로 deploy 하시면 됩니다.

## 파라미터 커스터마이징

 `config.json`에 GPT 모델과 파라미터가 들어 있습니다. 필요시 변경 가능합니다.

## 메인 스크립트

 `proto.py`
